Summary: Latitude vs. X

In this project, data was pulled from 689 cities which were selected at random from OpenWeatherMap API. For each city, data was pulled based for the longitude, latitude, cloudiness, humidity, wind speed, and max tempterature.

Once this data was obtained, a series of graphs were created using MatPlotLib to compare the four charecteristics; cloudiness, max temp, humidity, and wind speed, against the latitude of the city. The goal was to see which, if any, of these factors were effected by the cities proximity to the equator. This website shows the findings as well as a description of the graphs
